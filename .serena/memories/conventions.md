# Conventions

- Keep modules small: API schemas/routes in `src/api/`; model loading/prediction in `src/model/`; feature alignment in `src/features/`; SHAP/template summaries in `src/explain/`; drift in `src/monitor/`; training in `src/train/` or `kaggle_kernel/`.
- Preserve `/predict` as non-LLM deterministic scoring. Optional LLM work belongs to `/explain/llm` and must not change the real-time prediction contract.
- Serving payloads are sparse; `build_serving_frame()` zero-fills the IEEE feature set and accepts demo fields/aliases like `TransactionAmt`, `TransactionDT`, `ProductCD`, `card1`, `amount`, `hour`.
- Threshold-sensitive text must use the configured cost threshold, not a hard-coded `0.5`.
- Entry points use `if __name__ == "__main__"` for scripts only; API is imported by Uvicorn and has no `main()` function.
- Generated monitoring HTML/summary should be regenerated by `make monitor`; avoid hand-editing generated report output.
- Do not treat local `make train` artifacts as production replacement for the committed Kaggle-trained IEEE artifact.
